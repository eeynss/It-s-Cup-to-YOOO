# It’s Cup to Yoo – WS101 Final Project

## Project Overview
This project was created as our final requirement for WS101: Web System. It is a responsive and interactive website developed for **It’s Cup to Yoo**, a frozen yogurt, ice cream, and snack business.

The website was designed to showcase the brand’s products, menu, flavors, and customer experience through a modern and visually appealing interface. The project demonstrates our skills in front-end web development, responsive design, animations, and user interaction.

---

## Permission
This project was created for academic purposes only. Any logos, product images, and branding used in the website belong to **It’s Cup to Yoo** and are used solely for educational and presentation purposes.

---

## Objectives
- Demonstrate the ability to create a responsive website
- Apply modern web design and UI/UX principles
- Implement animations and interactive website elements
- Improve user experience across desktop, tablet, and mobile devices
- Showcase branding and menu offerings effectively

---

## Key Features
- Responsive layout for desktop, tablet, and mobile devices
- Light mode and dark mode theme switcher
- Interactive mobile navigation menu
- Smooth scrolling and transition animations
- Animated product slider and showcases
- Glassmorphism-inspired user interface
- Flavor and toppings showcase section
- Social media and contact integration
- Modern landing page design

---

## Technologies Used
- HTML5 – Semantic webpage structure
- CSS3 – Styling, animations, responsive layout, and effects
- JavaScript – Interactivity and functionality
- Responsive Web Design – Mobile-friendly layout techniques
- Git & GitHub – Host Version control and project management
